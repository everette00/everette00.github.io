Verbkit

Verbit is a declarive TypeScript API for building browser-based video games. The premise of the project is to reduce the burden of developing games by providing pre-built components common across many 2D video games. Moreover, Verbkit comes with an API that allows for more components to be built by users, so that they can create games Verbkit doesn't support out the box.

Verbkit took a declarative approach for its API to allow users to state the world and all the game objects involved with this world. These worlds are their own script that exists in the game directory.
